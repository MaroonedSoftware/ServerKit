Add `@maroonedsoftware/permissions-dsl` — a surface-syntax DSL and `pdsl` CLI for authoring `@maroonedsoftware/permissions` namespaces in `.perm` files. Includes an Ohm-based parser, semantic validator, TypeScript codegen, and a multi-file compiler driven by `permissions.config.json`.
